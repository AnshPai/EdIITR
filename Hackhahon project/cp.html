Welcome to the world of Competitive Programming

So, what do you think is Competitive Programming!!

Competitive programming is a mind sport usually held over the Internet or a local network, involving participants trying to program according to provided specifications. Contestants are referred to as sport programmers.

So, basically to start with competetive programming, generally called CP, you should know some basic programming language. There are various programming languages like C, C++, Java, Python and a lot more. If you are new to this programming world, you can get a headstart to coding from our site only. You can find material for learning coding from our homepage. There you will find various other resources as well so do check it out and explore.

Coming back to CP. So, I would assume that now you have atleast some basic knowledge of this somewhat new world to you. Now, we can look forward to competetive programming.

By now, you know that CP is a sport. Like every sport, even this has some rules which depend on your playground (we would soon be talking about this 'playground').

Some of these rules are
1. Whatever code you write, should be execultable xD. Offcourse!!!!
2. Now, your executable code should be executed in a specific time depending on the problem, which is clearly mention in your 'playground'. What this means is that your code should not take more than a specific time for compilation and execution.
3. Your code should not take more than given disk size.

So now we are going to talk about our playground. There are various online platform where you can play this sport. Following are the list of some of these most famous platforms where you can just login and start your journey.
The following are some online platforms for CP

  • Codechef

  • Codeforces

  • Hackerrank

  • Topcoder

  • HackerEarth

  • AtCoder

  • CS Academy

  • HackerBlocks

There are some online judges and problem archives as well which will help you grow as a competitive programmer.
Some of these are mentioned below
